year range extract and replacement

Usage

rngyear(object, ...)

Arguments

object
an FLComp object
value
a vector with max and min year range to replace the object info

Value

a vector object when extracting, or an FLComp object when replacing

Description

Methods to extract from FLComp objects the year range, or to replace its value.

Examples

data(ple4)
data(ple4.index)
rngyear(ple4)



minyear maxyear 
   1957    2008 



rngyear(ple4.index)



minyear maxyear 
   1985    2008
