age range extract and replacement

Usage

rngage(object, ...)

Arguments

object
an FLComp object
value
a vector with max and min age range to replace the object info

Value

a vector object when extracting, or an FLComp object when replacing

Description

Methods to extract from FLComp objects the age range, or to replace its value.

Examples

data(ple4)
data(ple4.index)
rngage(ple4)



min max 
  1  10 



rngage(ple4.index)



min max 
  1   8
