M bar range extract and replacement

Usage

rngmbar(object, ...)

Arguments

object
an a4aM object
value
a vector with max and min age range to replace the object info

Value

a vector object when extracting or an a4aM object when replacing

Description

Methods to extract from a4aM objects the M bar age range, or replace its value.

Examples

mod1 <- FLModelSim(model=~exp(-age-0.5))
mod2 <- FLModelSim(model=~1.5*k, params=FLPar(k=0.4))
m1 <- a4aM(shape=mod1, level=mod2)
rngmbar(m1)<-c(1,5)
rngmbar(m1)
